# Amazon S3

Amazon Simple Storage Service (Amazon S3) is an object storage service offering industry-leading scalability, data availability, security, and performance.

### Bucket: swasthx-bucket

**Purpose:**
- Central storage for all application assets
- Document and image repository

**Features:**
- Versioning disabled
- Bucket owner enforced ownership
- Static website hosting capabilities
- CORS configuration for web access

## Configuration Settings

### General Properties
- **Region**: Asia Pacific (Mumbai) `ap-south-1`
- **Bucket Versioning**: Disabled
- **MFA Delete**: Disabled

### Permissions
**Block Public Access**: Off (Public access is allowed via bucket policy)

### Bucket Policy
The following policy grants public read access (`s3:GetObject`) to all objects in the bucket:

```json
{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Effect": "Allow",
            "Principal": "*",
            "Action": "s3:GetObject",
            "Resource": "arn:aws:s3:::swasthx-bucket/*"
        }
    ]
}
```

### CORS Configuration
Cross-Origin Resource Sharing (CORS) is configured to allow access from the Swasthx domain and local development environments:

```json
[
    {
        "AllowedHeaders": [
            "*"
        ],
        "AllowedMethods": [
            "GET",
            "HEAD"
        ],
        "AllowedOrigins": [
            "https://*.swasthx.com",
            "https://c98rgke3sa.ap-south-1.awsapprunner.com",
            "http://localhost:*",
            "http://127.0.0.1:*"
        ],
        "ExposeHeaders": [],
        "MaxAgeSeconds": 3000
    }
]
```
